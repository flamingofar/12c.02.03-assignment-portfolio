Designing With Frameworks

This project is about frameworks, using different tools and implementing a framework. Here i went through different tools like pico, bootstrap and tailwind. In the project i utilized tailwind.

CSS frameworks

There are tons of CSS frameworks out there, and we got to play with Pico, Bootstrap and the brave could dip their hands in Tailwind. Pico and Bootstrap had one thing in common, the result looks really bootstrapy and generic. As Bootstrap are component based it is reasonably fast to build interfaces.

Tailwind

I went with tailwind for this project. As i discovered, tailwind and the component based frameworks, like Bootstrap, are really not compareable. Tailwind is a giant library of utility classes and inforces you to use a design system and implement that into the core of tailwind. The process of building this project got longer becouse of tailwind being utility bassed and not component based.
